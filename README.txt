Double click WaterJugProblem.cpp to start our program

	-This will create a table based on the two numbers the user inputs 
	-Numbers can't be greater then 99
	-No negatives
    -No letters

The first input it will ask the user to put in a number for the first jug.

Then it will ask the user to put a number for the second jug. 

Lastly will tell you the number of rows the table will have and output the table on the screen and create a file
called jugs that has the table in it.
